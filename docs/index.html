Maze Solve

Maze Solve is an implementation of Dijkstra's Path finding algorithm, using a Binary Heap as a priority queue. It can take an image like this...

and will give you the solution


It was built with modularity in mind; i.e. one could write other path finding algorithms with the same framework easily.

This was a fun and challenging project to take on. It was my first (relatively) large project, and I learned a lot about time/space complexity, debugging, and python in the process.
